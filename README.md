# Principles of Information Security – Programming Assignments

Python implementations of programming exercises from the **Principles of Information Security** course.

---

## Assignment List

| No | Question | File |
|----|---------|------|
| 01 | Create a provably secure PRG based on the hardness of the Discrete Logarithm Problem (DLP). Input: F, G. Output: PRG. | `q1_secure_prg_dlp.py` |
| 02 | Create a Pseudo Random Function from the PRG in Q1 | `q2_prf_from_prg.py` |
| 03 | Use PRF in Q2 to build a CPA secure encryption scheme | `q3_cpa_from_prf.py` |
| 04 | Design CPA secure scheme with LD(double lengthening), CBC, OFB, RC(Randomized Counter) and PRF or AES | `q4_cpa_block_ciphers.py` |
